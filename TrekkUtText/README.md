# Trekk ut text fra en JSON-fil

## Kom i gang:
### JSON til tekst
    ```bash python3 JSONToText <FraFil> <TilFil>```
### Tekst til JSON
    ```bash python3 TextToJSON <JsonStrukturFil> <TextFil> <TilFil>```
