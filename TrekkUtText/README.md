# Trekk ut tekster fra JSON og bygg opp JSON fra tekst

## Kom i gang:

### Forutsetning:

- Installer python

### JSON til tekst
    
    bash python3 JSONToText <FraFil> <TilFil>
### Tekst til JSON
    
    bash python3 TextToJSON <JsonStrukturFil> <TextFil> <TilFil>
