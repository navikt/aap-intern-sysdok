# språkfil konvertering

For å konvertere *JSON* språkfil til .txt som kan sendes til oversettere,
Har vi laget 2 phyton scripts som koverterer til en tekstfil og som konverterer tilbake.

[JSON til tekstfil](https://github.com/navikt/utvikler_aap_innbygger/blob/main/TrekkUtText/JSONToText.py) og [Språkfil.txt til JSON](https://github.com/navikt/utvikler_aap_innbygger/blob/main/TrekkUtText/TextToJSON.py)
[Beskrivelse for hvordan finner du her](https://github.com/navikt/utvikler_aap_innbygger/blob/main/TrekkUtText/README.md).

## Forretningsbehov
Filen skal eksporteres til et format som kan åpnes i word (.txt kan åpnes i word).
